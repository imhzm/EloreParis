## Required owner inputs before public commerce

- Official SVG logo/mark and confirmation of the canonical palette version.
- Product packshots, variants, dimensions, alt text, and usage rights.
- Desktop and mobile Hero crops approved for publication.
- Actual packaging/gifting assets and the operations they truthfully represent.
- Written approval for any generated concept image promoted beyond a gated preview.
