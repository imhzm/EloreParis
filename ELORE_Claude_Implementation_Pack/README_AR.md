# حزمة تنفيذ ELORE Paris باستخدام Claude

## المحتويات

- `CLAUDE.md`: Roadmap كاملة، مع معايير التصميم والتنفيذ والـ QA وPrompts جاهزة.
- `reference/elore-home-concept.png`: الصورة المرجعية المعتمدة.
- `design-tokens.json`: لوحة الألوان، المسافات، الحواف، والحركة بصيغة قابلة للقراءة آليًا.

## الاستخدام

1. انسخ `CLAUDE.md` إلى جذر مشروعك.
2. أبقِ مجلد `reference` بجواره.
3. افتح المشروع في Claude Code أو ارفع الملفات إلى Claude.
4. استخدم الـ Master Prompt داخل نهاية `CLAUDE.md`.
5. نفّذ Milestone واحدة في كل مرة.

لا تستخدم الصورة المرجعية كصورة واحدة بدل بناء الواجهة. الهدف هو تحويلها إلى نظام مكونات حقيقي ومتجاوب.
